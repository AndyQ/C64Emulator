Download VICE for Mac OS X systems

You can download the latest VICE distributions for Mac OS X systems here. Some distributions may not be available at the release date.

The following (full-featured) ports are available on Mac OS X:

  • Cocoa - The native port (Preferred version)
  • X11/Gtk+ - Running on X11 with Gtk+
  • X11/Xaw - Running on X11 with Xaw
  • SDL - Using the SDL framework

Source

  • Tarball: vice-2.3.tar.gz

Cocoa

  • Power PC or Intel Mac
  • Universal Binary for Mac OS X >= 10.5: vice-macosx-cocoa-i386+ppc-10.5-gcc40-2.3.dmg
    Preferred version.
  • Universal Binary for Mac OS X >= 10.4: vice-macosx-cocoa-i386+ppc-10.4-gcc40-2.3.dmg
    This version runs on 10.4 but has limited features in the resource inspector than the 10.5 one.
  • Universal Binary for Mac OS X >= 10.6: vice-macosx-cocoa-i386+x86_64-10.6-clang-2.3.dmg
    This version runs on 10.6 on Intel only, but has 32 and 64 Bit version (clang compiler)
  • Universal Binary for Mac OS X >= 10.6: vice-macosx-cocoa-i386+x86_64-10.6-gcc42-2.3.dmg
    This version runs on 10.6 on Intel only, but has 32 and 64 Bit version (gcc compiler)

X11

  • Power PC or Intel Mac
  • Mac OS X >= 10.4
  • Installed X11.app from Extra Installs on Mac OS X Install DVD.
  • Universal Binary for Gtk+ UI: vice-macosx-gtk-i386+ppc-10.4-gcc40-2.3.dmg
  • Universal Binary for X11 Athena Widgets UI: vice-macosx-x11-i386+ppc-10.4-gcc40-2.3.dmg

SDL

  • Power PC or Intel Mac
  • Mac OS X >= 10.4
  • Installed SDL.framework from www.libsdl.org
  • Universal Binary for SDL UI: vice-macosx-sdl-i386+ppc-10.4-gcc40-2.3.dmg

The files are taken from http://www.zimmers.net/anonftp/pub/cbm/crossplatform/emulators/VICE.

Old binary distributions

Binaries of old releases can be found on http://www.zimmers.net/anonftp/pub/cbm/crossplatform/emulators/VICE.

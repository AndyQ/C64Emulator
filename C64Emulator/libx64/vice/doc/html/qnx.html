Download VICE for QNX systems

You can download the latest VICE distributions for QNX systems here. Some distributions may not be available at the release date.

Source distributions

  • Tarball: vice-2.3.tar.gz

Binary distributions

  • Binary for x86 QNX 4.x systems: vice-2.3-qnx4.tar.F. to be installed with the QNX 4.x installer.
  • Binary for x86 QNX 6.x systems: VICE-2.3-x86-public.qpr. To be installed with the QNX 6.x installer.
  • Binary for armle QNX 6.x systems: SDLVICE-2.3-armle-public.qpr. To be installed with the QNX 6.x installer.
  • Binary for mipsle QNX 6.x systems: SDLVICE-2.3-mipsle-public.qpr. To be installed with the QNX 6.x installer.
  • Binary for ppcbe QNX 6.x systems: SDLVICE-2.3-ppcbe-public.qpr. To be installed with the QNX 6.x installer.
  • Binary for shle QNX 6.x systems: SDLVICE-2.3-shle-public.qpr. To be installed with the QNX 6.x installer.

The files are taken from http://www.zimmers.net/anonftp/pub/cbm/crossplatform/emulators/VICE.

Old binary distributions

Binaries of old releases can be found on http://www.zimmers.net/anonftp/pub/cbm/crossplatform/emulators/VICE.
